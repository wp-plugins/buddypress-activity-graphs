Allows visualisations to be shown for how users use your buddypress site

== Description ==

This plugin allows for BuddyPress to be extended (on each user profile) by showing a variety of different forms of activities visualised

== Installation ==

You can download and install using the built in WordPress plugin installer. 
If you download manually, make sure the folder is uploaded to "/wp-content/plugins/".

Activate the plugin in the "Plugins" admin panel using the "Activate" link.
